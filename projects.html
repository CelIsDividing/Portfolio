FPGA Visualization Tool

I developed a comprehensive visualization framework that transforms raw FPGA implementation data into intuitive graphical representations. The system parses post-routing reports and timing analyses to generate interactive visualizations of signal pathways, clearly distinguishing between critical paths and regular interconnects with color-coded schemes. I implemented directional flow diagrams that trace signal propagation through LUTs, flip-flops, and routing switches, making timing constraints and potential bottlenecks immediately apparent. The tool also renders dynamic bounding boxes around related logic clusters, highlighting placement efficiency and resource utilization patterns. Most notably, it constructs and displays conflict graphs where competing signal routes for limited routing resources are represented as connected nodes, allowing me to quickly identify routing congestion hotspots and optimize resource allocation decisions. This visualization suite has proven invaluable for debugging complex timing issues and understanding the spatial characteristics of my FPGA designs.

Music Genre Recognition Neural Network

I built a deep learning model that classifies music into genres using audio features extracted from spectrograms. The network employs convolutional layers to identify patterns in frequency-time representations of audio signals, achieving over 85% accuracy across 10 genre categories. I implemented data augmentation techniques and cross-validation to improve generalization and reduce overfitting. The model was trained on a dataset of 10,000 labeled tracks and deployed as a web application using Flask and TensorFlow.js for real-time genre prediction.
